API 接口支付示例

接口列表

POST /api/payment/create

创建支付订单，返回二维码或支付链接

请求参数（JSON）：

{
    "pay_type": "alipay",  // 支付方式：alipay, wxpay, qqpay
    "amount": 0.01,        // 金额（元）
    "name": "商品名称"      // 商品名称
}

GET /api/payment/query?out_trade_no=xxx

查询订单状态

POST /api/payment/notify

支付回调通知（由 EPay 服务器调用）
